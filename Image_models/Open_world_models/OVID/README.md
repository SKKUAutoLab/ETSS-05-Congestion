# OVID

## 1. To download the pretrained weight, run the following script:
```shell
bash scripts/download_weight.sh
```

## 2. To train and test the model for ShanghaiTech and FSC147 datasets, run the following scripts:
```shell
bash scripts/train_shanghaitech.sh
bash scripts/train_fsc147.sh
bash scripts/test_shanghaitech.sh
bash scripts/test_fsc147.sh
```
